# LearningSystem

LearningSystem

LearningSystem is a minimal AI Learning System MVP that parses resumes,
generates short learning plans, and tracks simple mastery per skill.

This repository contains a lightweight FastAPI backend, a static UI, and
sample placeholders to demo upload, plan generation, and assessment flows.

Run instructions:

- Install dependencies: `pip install -r requirements.txt`
- Start the server: `uvicorn main:app --reload`
